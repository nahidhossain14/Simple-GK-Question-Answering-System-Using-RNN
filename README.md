# Simple GK Question Answering System using RNN

A simple Natural Language Processing (NLP) project built with PyTorch.

## Features

- Text Tokenization
- Vocabulary Building
- Custom Dataset
- RNN Model
- Question Answer Prediction

## Technologies

- Python
- PyTorch
- Pandas
- NumPy

## Dataset

General Knowledge Question Answer Dataset

## How to Run

1. Upload dataset
2. Run all notebook cells
3. Ask questions
